导入github中的项目

Feb 17 2019 a minute read (About 213 words)

准备

  • 安装 homebrew

    • shadowsocks + profixier 加速安装速度
      /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

  • homebrew 安装 git
    brew install git

  • 设置变量
    git config --global user.name 'zz'
    git config --global user.email '656453@qq.com'

  • 创建版本库
    git init
    ls -ah

导入项目

  • 创建ssh key^liaoxuefeng
    ssh-keygen -t rsa -C "youremail@example.com"
  • 将公钥 Users/zz/.ssh/id_rsa.pub 登记在github的设置里
  • 查看关联的远程库
    git remote -v
  • 删除关联的远程库
    git remote rm origin
  • 关联远程库
    git remote add origin git@github.com:zoo-keeper/leetcode.git
  • 导入远程库
    git pull origin master

测试

###在文件管理器修改/删除文件

  • 提交到当前工作区
    git add .
  • 提交到本地仓库
    git commit -m "注释"
  • 提交到远程库
    git push origin master
